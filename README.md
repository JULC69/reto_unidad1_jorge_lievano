# reto_unidad1_jorge_lievano

## Descripción

Proyecto desarrollado para el Reto Unidad 1 de análisis de datos utilizando:

- VSCode
- Python
- Jupyter Notebook
- NumPy
- Pandas
- SQLite
- Anaconda

El proyecto realiza actividades de exploración, limpieza, transformación y análisis de un conjunto de datos de ventas de una empresa tecnológica.

## Objetivos

### Notebook 01 - NumPy

- Carga del dataset de ventas.
- Conversión de columnas a arrays NumPy.
- Operaciones vectorizadas.
- Producto matricial.
- Estadísticas descriptivas.

### Notebook 02 - SQL Discovery

- Carga de datos con Pandas.
- Creación de una base de datos SQLite en memoria.
- Importación del DataFrame a SQLite.
- Ejecución de consultas SQL.
- Creación e inserción de datos en tablas adicionales.

### Notebook 03 - Pandas Pipeline

- Detección automática de codificación.
- Limpieza de datos.
- Validación de calidad.
- Transformación de datos.
- Preparación del dataset final.
